DigDug

DigDug é um projeto baseado no famoso jogo retro de mesmo nome, onde o objetivo é cavar túneis, derrotar inimigos e coletar tesouros. Este projeto foi desenvolvido utilizando a linguagem de programação Python, empregando técnicas de inteligência artificial (IA) para criar inimigos e desafios interativos dentro do jogo. É um projeto simples, mas desafiador, perfeito para quem quer explorar conceitos de IA e programação de jogos.
